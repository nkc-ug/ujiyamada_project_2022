<h2>Githubのルールについて</h2>
  <h3>コメントの書き方</h3>
  <ul>
    <li>日本語で書く</li>
    <li>プログラムの何を変えたのか詳細に書く</li>
    <li>プログラムを改変したところだけにコメントがくるようにする</li>
  </ul>
  <h3>Githubのフォルダ構成の変更について</h3>
  <ul>
    <li>プロジェクトファイルの構成を変更したいときはリーダーに報告する</li>
    <li>バックエンド、フロントエンド専用のファイルをそれぞれ作成し、管理する</li>
  </ul>
